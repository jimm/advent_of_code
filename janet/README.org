You'll have to install spork.
: jpm install spork

The simplest way to run a day's answers is to use the `../aoc` script.

When running a day as a script, the first optional arg is "-t" to set
testing true. The final arg is the part number.

#+begin_src sh
  janet/2024/day01.janet -t 1
#+end_src
